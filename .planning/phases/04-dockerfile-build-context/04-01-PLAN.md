Rewrite the Dockerfile with these changes:

**Base stage** (`FROM php:8.3-fpm AS base`):
1. Add `gosu` to the `apt-get install` list (research A1 — available in Debian Bookworm). This is required by entrypoint.sh line 57 (`exec gosu`).
2. Add `opcache` to the `docker-php-ext-install` extension list (per DOCKER-04). OPcache is bundled with PHP 8.3, no configure step needed (research A2).

**Builder stage** (`FROM base AS builder`):
3. Optimize layer caching (per D-05, D-06): Replace the single `COPY . .` with dependency-first copy pattern:
   - `COPY composer.json composer.lock ./`
   - `RUN composer install --no-dev --no-interaction --prefer-dist --optimize-autoloader`
   - `COPY package*.json ./`
   - `RUN npm ci`
   - `COPY . .`
   - `RUN npm run build`

   Use `npm ci` instead of `npm install` for reproducible builds from lockfile. `npm ci` is faster and respects `package-lock.json` exactly.

**Production stage** (`FROM base AS production`):
4. Keep defense-in-depth `rm -rf` per D-04: `RUN rm -rf node_modules .git .github tests`
5. Add ENTRYPOINT and CMD directives (per D-07, DOCKER-03):
   - `ENTRYPOINT ["/var/www/docker/prod/entrypoint.sh"]` — absolute path per research Pitfall 4
   - `CMD ["php-fpm"]` — default command passed to entrypoint

   ENTRYPOINT + CMD interaction: Docker runs `/var/www/docker/prod/entrypoint.sh php-fpm`. The entrypoint checks `$1 = "php-fpm"` and delegates to `docker-php-entrypoint`. For override (e.g., `docker run <image> artisan migrate`), Docker runs `/var/www/docker/prod/entrypoint.sh artisan migrate`, and the entrypoint calls `exec gosu www-data artisan migrate`.

   Per D-08: No `USER` directive — the entrypoint handles privilege drop via gosu. A USER directive would break the entrypoint's root-level chown operations.

The final Dockerfile structure should be:

```
FROM php:8.3-fpm AS base
    apt-get install ... gosu
    docker-php-ext-install ... opcache
    WORKDIR /var/www
    COPY --from=composer:2 /usr/bin/composer /usr/bin/composer

FROM base AS builder
    COPY composer.json composer.lock ./
    RUN composer install ...
    COPY package*.json ./
    RUN npm ci
    COPY . .
    RUN npm run build

FROM base AS production
    COPY --from=builder /var/www /var/www
    RUN rm -rf node_modules .git .github tests
    ENTRYPOINT ["/var/www/docker/prod/entrypoint.sh"]
    CMD ["php-fpm"]
```
