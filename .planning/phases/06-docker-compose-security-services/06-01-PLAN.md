**B) Queue health check (per D-07, D-08, COMPOSE-05):** Add a healthcheck block to the queue service using a process-aliveness check:

```yaml
  queue:
    ...
    healthcheck:
      test: ["CMD-SHELL", "pgrep -f 'queue:work'"]
      interval: 30s
      timeout: 5s
      retries: 3
      start_period: 10s
```

Use `CMD-SHELL` (not `CMD`) because `pgrep -f 'queue:work'` uses shell quoting for the `-f` pattern match. The `start_period` gives the worker 10 seconds to boot before health checks begin failing.
</action>
